Forgot to create a branch


If you begin working, but you are still on main, don’t panic! Changes aren't “on” a branch until they are committed.

Your working directory and staging area are attached to HEAD, not a branch. So, you can change branches, and any changes that are not committed will come with you.

  1. Type git status, and notice changes in your working directory
  2. Create a new branch with git switch -c BRANCH-NAME
  3. Type git status again, to notice which branch you are on, and your working directory
